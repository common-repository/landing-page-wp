== Description ==

**Landing Page Plugin**

The #1 Landing Page & Coming Soon plugin help you to manage your wordpress site and display responsive coming soon page or maintenance mode page on one click.  Landing page is provide control of your coming soon website or under maintenance website, it's also Redirect your wordpress website to any url using this plugin. 

https://www.youtube.com/watch?v=sXyRu9Htelo&feature=youtu.be

In this plugin you can find two types of mode - first one is landing page mode and second one is redirect mode.

== Installation ==

1. Upload the entire `landing-page-wp` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Landing Page Panels menu to customize this plugin.
4. Finally Enjoy it.
